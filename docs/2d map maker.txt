### 2. **2D Parchment Map Editor (In-Game UI)**
Create a `MapEditor.tscn` (CanvasLayer root):
- **Canvas:** `TextureRect` (size 1024x1024) with `Image` backend for layers.
  - **Height Layer:** Grayscale Image (black=sea, white=mountains).
  - **Biome Layer:** RGB Image (R=grass, G=rock, B=water).
- **Tools Panel:** `VBoxContainer` with buttons: Raise, Lower, Smooth, Biome Picker (color buttons).
- **Preview:** `SubViewport` → `TextureRect` for live 2D view.

**GDScript (attach to MapEditor Node):**
```gdscript
extends CanvasLayer
@onready var canvas: TextureRect = $Canvas
@onready var height_img: Image = Image.create(1024, 1024, false, Image.FORMAT_RF)
@onready var biome_img: Image = Image.create(1024, 1024, false, Image.FORMAT_RGB8)
var brush_size: int = 20
var strength: float = 0.1
var mode: String = "raise"  # raise, lower, smooth, biome
var brush_color: Color = Color.GREEN  # For biomes

func _ready():
    height_img.fill(Color.BLACK)  # Sea level
    update_canvas()

func _input(event):
    if event is InputEventMouseButton and event.pressed and canvas.get_rect().has_point(event.position):
        paint(event.position, event.button_index == MOUSE_BUTTON_RIGHT)  # Left: paint, Right: smooth/erase
    elif event is InputEventMouseMotion and Input.is_mouse_button_pressed(MOUSE_BUTTON_LEFT):
        paint(event.position)

func paint(pos: Vector2, smooth: bool = false):
    var local_pos = canvas.to_local(pos)
    var center = Vector2i(local_pos) / canvas.size * 1024
    for y in brush_size:
        for x in brush_size:
            var px = center + Vector2i(x - brush_size/2, y - brush_size/2)
            if px.x >= 0 and px.x < 1024 and px.y >= 0 and px.y < 1024:
                match mode:
                    "raise": height_img.set_pixel(px.x, px.y, height_img.get_pixel(px.x, px.y) + Color(strength,0,0))
                    "lower": height_img.set_pixel(px.x, px.y, height_img.get_pixel(px.x, px.y) - Color(strength,0,0))
                    "smooth":
                        var avg = (height_img.get_pixel(px.x-1,px.y) + height_img.get_pixel(px.x+1,px.y) +
                                   height_img.get_pixel(px.x,px.y-1) + height_img.get_pixel(px.x,px.y+1)) / 4.0
                        height_img.set_pixel(px.x, px.y, height_img.lerp(height_img.get_pixel(px.x, px.y), avg, 0.1))
                    "biome": biome_img.set_pixel(px.x, px.y, brush_color)
    height_img.clamp_image()  # 0-1 grayscale
    update_canvas()

func update_canvas():
    var combined = Image.create(1024, 1024, false, Image.FORMAT_RGBA8)
    for y in 1024:
        for x in 1024:
            var h = height_img.get_pixel(x, y).r
            var b = biome_img.get_pixel(x, y)
            combined.set_pixel(x, y, Color(b.r, b.g, b.b, h))  # Height as alpha for preview
    canvas.texture = ImageTexture.create_from_image(combined)

func generate_3d():
    height_img.save_exr("user://heightmap.exr")  # Best for Terrain3D
    biome_img.save_png("user://biomes.png")  # Control map
    get_tree().change_scene_to_file("res://VTTScene.tscn")
```
- **Brush UI:** Add `HSlider` for size/strength, `OptionButton` for modes, `ColorPickerButton` for biomes.

**Parchment Style Overlay:**
- Add `ColorRect` (full screen) behind canvas with `ShaderMaterial`.
- **Parchment.shader** (new ShaderMaterial → CanvasItem):
```glsl
shader_type canvas_item;
uniform sampler2D parchment_tex : source_color;
uniform float curl_strength : hint_range(0, 0.1) = 0.05;
uniform float stain_intensity : hint_range(0, 1) = 0.7;

void fragment() {
    vec2 uv = SCREEN_UV;
    // Curl edges
    uv.x += sin(uv.y * 10.0) * curl_strength * (1.0 - uv.y);
    uv.y += cos(uv.x * 10.0) * curl_strength * (1.0 - uv.x);
    // Noise for crumple/stains
    float noise = texture(SCREEN_TEXTURE, uv + vec2(TIME * 0.1, 0)).r;
    vec4 parchment = texture(parchment_tex, uv);
    COLOR = texture(TEXTURE, uv) * parchment * (1.0 - noise * stain_intensity * (1.0 - uv.x) * (1.0 - uv.y));  // Darken edges
    COLOR.a = 1.0;
}
```
- Assign stained parchment PNG. Tweak for yellowed paper, ink bleed.

**Procedural Seed (Bonus):** Add noise gen button using `FastNoiseLite` for continents (like previous convo).

### 3. **3D Terrain Generation & VTT Scene**
- **VTTScene.tscn:** `Node3D` root + `Terrain3D` node.
  - Inspector: **Data > Import Heightmap** → Load `user://heightmap.exr`.
  - **Textures:** Assign 4-8 fantasy mats (grass, rock, snow, water). Use biome PNG as **Control Map** (R/G/B/A channels blend textures).
  - Scale: 1000x1000 units for tabletop size; height 0-200m.
- **FPS Camera:** `CharacterBody3D` + `Camera3D` + `CollisionShape3D`.
```gdscript
# FPSController.gd
extends CharacterBody3D
var speed = 5.0
func _physics_process(delta):
    var input = Input.get_vector("ui_left", "ui_right", "ui_up", "ui_down")
    velocity = Vector3(input.x, 0, input.y) * speed
    move_and_slide()
```
- **Minis/Tokens:** Draggable `RigidBody3D` models (free D&D minis on Sketchfab). Raycast from mouse for placement.
- **Fog of War:** `Decal` or `CanvasModulate` shader masking unexplored areas.
- **Grid:** `MeshInstance3D` with grid plane + snap logic (divide terrain into 1m hex/squares).
